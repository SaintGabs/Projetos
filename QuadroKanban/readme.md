# Configuração
---

### 1. Criar Conta ou Logar
Acesse o site oficial da [WeatherAPI](https://www.weatherapi.com) e clique em **Login** ou **Signup** (caso ainda não tenha conta).

---

### 2. Copiar a API Key
No campo **API Key**, você verá uma sequência de letras e números. Clique no ícone de copiar.

<img width="575" height="97" alt="image" src="https://github.com/user-attachments/assets/df5598eb-d580-457d-92fc-bbd8cb5702b5" />

---

### 3. Configurar a Aplicação
Cole a chave copiada no campo abaixo dentro do sistema para liberar o acesso aos dados meteorológicos:

**KEY:** `COLE_SUA_CHAVE_AQUI`
